### Recommended refactor skeleton

```tsx
<section role="region" aria-labelledby="evidence-dashboard-title" className="…">
  <h2 id="evidence-dashboard-title">แดชบอร์ดหลักฐาน</h2>
  <span role="status" aria-live="polite" lang="th">{dataSourceLabel}</span>

  <dl className="grid …">
    <div role="group" aria-labelledby="kpi-total-label">
      <dt id="kpi-total-label">รวมทั้งหมด</dt>
      <dd aria-label={`รวม ${total.toLocaleString("th-TH")} รายการ`}>
        {total.toLocaleString("th-TH")}
      </dd>
    </div>
    {/* …more KPIs… */}
  </dl>

  <svg role="img" aria-labelledby="spark-title spark-desc" width={sparkW} height={sparkH}>
    <title id="spark-title">หลักฐาน 7 วันล่าสุด</title>
    <desc id="spark-desc">สูงสุด {max} รายการ ต่ำสุด {min} รายการ รวม {sum}</desc>
    <polyline points={poly} fill="none" stroke="currentColor" strokeWidth={2} />
  </svg>

  <table>
    <caption className="sr-only">สรุปจำนวนหลักฐานต่อ ISP</caption>
    <thead>
      <tr>
        <th scope="col" aria-sort={sortBy === "isp" ? (sortDir === "asc" ? "ascending" : "descending") : "none"}>
          <button type="button" onClick={() => toggleSort("isp")}>ISP</button>
        </th>
        <th scope="col" aria-sort={sortBy === "count" ? (sortDir === "asc" ? "ascending" : "descending") : "none"}>
          <button type="button" onClick={() => toggleSort("count")}>จำนวน</button>
        </th>
      </tr>
    </thead>
    <tbody>
      {sortedRows.length === 0 ? (
        <tr><td colSpan={2} role="status">ไม่มีข้อมูล</td></tr>
      ) : sortedRows.map((r) => (
        <tr key={r.isp}><th scope="row">{r.isp}</th><td>{r.count.toLocaleString("th-TH")}</td></tr>
      ))}
    </tbody>
  </table>
</section>
```

### Quick wins (≤ 30 min)
1. Promote dashboard & KPI titles to real headings (`<h2>`, `<h3>` / `<dt>`).
2. Add `role="region"` + `aria-labelledby` to the container.
3. Convert the table from `<div>` grid to a native `<table>` with `<th scope>` and an `aria-sort` state.
4. Give the sparkline `role="img"` + `<title>` + `<desc>` with min/max/sum text.
5. Add `role="status"` / `aria-live="polite"` to the data-source badge and unavailable banner.
6. Verify contrast: amber-900 on amber-50 (light) and green-200 on gray-900/30 (dark) — bump to AA-safe pairs.
7. Add visible focus rings on every interactive element.
