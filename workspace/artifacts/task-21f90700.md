ภาษาไทยเป็นภาษาที่อยู่ในตระกูล **ไท–กะได (Tai‑Kadai)** โดยเฉพาะอยู่ในสาขา **ไท (Tai)** ซึ่งเป็นสาขาย่อยของไท–กะได ภาษาใกล้เคียง ได้แก่ ภาษาลาว ภาษาชีน ภาษาพม่าในกลุ่มไท เป็นต้น  

**ลักษณะสำคัญของภาษาไทย**

| หัวข้อ | รายละเอียด |
|--------|------------|
| **ตระกูลภาษา** | ไท–กะได (Tai‑Kadai) → สาขาไท (Tai) |
| **ภาษาราชการ** | ภาษาทางการของประเทศไทย |
| **จำนวนผู้พูด** | ประมาณ 60–70 ล้านคน (ในประเทศไทย และชุมชนพลัดถิ่น) |
| **ระบบเสียง** | เป็น **ภาษาเสียงวรรณยุกต์** (tonal language) มี 5 เสียงวรรณยุกต์ (เสียงสูง เสียงต่ำ เสียงขึ้น เสียงลง เสียงกลาง) |
| **โครงสร้างประโยค** | เป็น **ภาษาแบบแยกคำ (isolating / analytic)** คำคงที่ไม่มีการผันรูป มักใช้คำสันธานและคำบ่งชี้เพื่อแสดงความสัมพันธ์ทางไวยากรณ์ |
