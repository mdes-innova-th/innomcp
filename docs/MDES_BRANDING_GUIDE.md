# คู่มือแบรนด์ MDES สำหรับ INNOMCP

## สี (Colors)
- **MDES Primary**: `#1a3c6e` (น้ำเงินราชการ) – สีหลักของแถบนำทางและองค์ประกอบสำคัญ
- **MDES Primary Light**: `#2d5a9e` – ใช้สำหรับสถานะ hover, เส้นขอบ หรือพื้นหลังอ่อน
- **MDES Accent**: `#c8973e` (ทองไทย) – สีเสริมเพื่อเพิ่มความโดดเด่น เช่น เน้นตัวเลขสำคัญหรือไอคอน
- **Text Primary**: ใช้ `#111827` (dark mode) หรือ `#ffffff` (light mode) ขึ้นอยู่กับธีม

## โลโก้ (Logo)
- **เครื่องหมาย**: ใช้ emoji 🇹🇭 ตามด้วยข้อความ “MDES” (Thai Sarabun Bold, สีขาวบนพื้น Primary)
- **คำบรรยายใต้โลโก้**: “ศูนย์ MCP ภาครัฐ” (font size 14px, สี Primary Light)
- **ชื่อเต็มหน่วยงาน**: “กระทรวงดิจิทัลเพื่อเศรษฐกิจและสังคม” (แสดงเฉพาะในส่วนติดต่ออย่างเป็นทางการ)

## การพิมพ์ (Typography)
- **ฟอนต์ภาษาไทย**: Noto Sans Thai, Sarabun (ใช้ Sarabun สำหรับหัวข้อหลัก)
- **ฟอนต์ภาษาอังกฤษ/ตัวเลข**: Inter
- **ขนาดตัวอักษรพื้นฐาน**: 15px (body) โดยมี line-height 1.7 เพื่อความสบายตาสำหรับเนื้อหาภาษาไทย
- **หัวข้อ**: ใช้ขนาด 20–28px ตามลำดับชั้นของหัวเรื่อง

## แนวทางส่วนประกอบ UI (Component Guidelines)
- **ปุ่มหลัก**: พื้นหลัง `bg-[#1a3c6e]` ตัวอักษรสีขาว มีมุมโค้งเล็กน้อย (rounded-md) และ hover lighten เป็น `#2d5a9e`
- **MDES Brand Pill**: ใช้สี indigo accent สำหรับแท็กสถานะ เช่น “MDES Active” หรือป้ายแบนเนอร์
- **Status Ribbon**: แถบสถานะด้านบนสุดใช้พื้นหลัง indigo เมื่อระบบกำลังเชื่อมต่อกับ MDES

## เสียงและโทน (Voice & Tone)
- **ความเป็นราชการที่เข้าถึงได้**: ใช้ภาษาไทยอย่างเป็นกันเองแต่คงความเป็นทางการ ไม่ใช้ศัพท์เทคนิคมากเกินไป
- **ยึดภาษาไทยเป็นหลัก**: คำอธิบาย เมนู และข้อความ UI ควรเป็นภาษาไทยก่อน หากจำเป็นให้ตามด้วยวงเล็บภาษาอังกฤษ
- **คำทางเทคนิค**: ใช้ “AI” (ไม่เขียนว่า “เอไอ”) แต่คำว่า “machine learning” ควรอธิบายเป็นไทยว่า “การเรียนรู้ของเครื่อง” พร้อมวงเล็บ ML
- **ความเป็นมิตร**: ใช้คำลงท้ายอย่างสุภาพ เช่น “ขออภัย” “เพื่อความสะดวก” และหลีกเลี่ยงประโยคคำสั่งที่แข็งกระด้าง
