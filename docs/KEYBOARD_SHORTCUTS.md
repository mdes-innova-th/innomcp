# คีย์ลัด INNOMCP

## การสนทนา
| คีย์ | การทำงาน |
|------|----------|
| Enter | ส่งข้อความ |
| Shift+Enter | ขึ้นบรรทัดใหม่ |
| Escape | หยุด AI |
| / | เปิดเมนูคำสั่ง Slash |

## การนำทาง
| คีย์ | การทำงาน |
|------|----------|
| Ctrl+K | เปิด Command Palette |
| Ctrl+/ | ดูคีย์ลัดทั้งหมด |
| Ctrl+N | แชทใหม่ |
| Ctrl+B | เปิด/ปิด Sidebar |

## Workspace
| คีย์ | การทำงาน |
|------|----------|
| Ctrl+W | เปิด/ปิด Workspace Panel |
| Ctrl+E | Export บทสนทนา |

## ทั่วไป
| คีย์ | การทำงาน |
|------|----------|
| ? | Help คีย์ลัด (เมื่อ input ว่าง) |
| Ctrl+, | เปิดการตั้งค่า |
