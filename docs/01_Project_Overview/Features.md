# ฟีเจอร์หลัก (Key Features)

InnoMCP มาพร้อมกับความสามารถที่ครบครันสำหรับการทำงานยุคใหม่

## 1. ระบบแชทอัจฉริยะ (Smart Chat Interface)
- **Real-time Streaming**: พิมพ์โต้ตอบกับ AI ได้ลื่นไหลไม่มีสะดุด
- **Multi-Model Support**: สลับระหว่างโมเดล AI ต่างๆ ได้ทันที (DeepSeek, Llama3, Mistral)
- **Code Highlighting**: แสดงโค้ดสวยงามพร้อมปุ่ม Copy

## 2. เครื่องมือขั้นเทพ (Advanced Tools via MCP)
ระบบ MCP ช่วยให้ AI ทำงานได้มากกว่าแค่คุย:
- **Web Search**: ค้นหาข้อมูลล่าสุดจากอินเทอร์เน็ต
- **File Reader**: อ่านไฟล์ PDF, txt และสรุปเนื้อหา
- **Image Intelligence**: ทำ OCR แปลงภาพเป็นข้อความผ่าน Tesseract.js
- **Chart Generation**: สร้างกราฟและชาร์ตสวยๆ จากข้อมูล

## 3. ระบบจัดการความรู้ (Knowledge Management)
- **Chat History**: บันทึกบทสนทนาย้อนหลังลง MariaDB
- **User Sessions**: ระบบ Login/Logout ปลอดภัย
- **Theme Customization**: ปรับแต่งธีมสีได้ตามใจชอบ

## 4. โครงสร้างพื้นฐาน (Infrastructure)
- **Dockerized**: ติดตั้งง่ายด้วย Docker Compose รันทีเดียวจบ
- **Health Checks**: มีสคริปต์ตรวจสอบสถานะระบบอัตโนมัติ
