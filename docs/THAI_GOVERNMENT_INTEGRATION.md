# คู่มือการบูรณาการ INNOMCP สำหรับหน่วยงานภาครัฐไทย

## ภาพรวม
INNOMCP เป็นแพลตฟอร์มปัญญาประดิษฐ์กลางของรัฐบาลไทย พัฒนาโดยกระทรวงดิจิทัลเพื่อเศรษฐกิจและสังคม (MDES) เพื่อเป็น **AI Layer** สำหรับบริการดิจิทัลภาครัฐทั้งหมด แพลตฟอร์มนี้เชื่อมต่อข้อมูลสำคัญจากหลายหน่วยงานด้วยมาตรฐาน MCP (Model Context Protocol) ทำให้ระบบราชการสามารถเรียกใช้ AI ได้อย่างปลอดภัยและมีประสิทธิภาพ

## Government APIs ที่รองรับ (ผ่าน MCP Tools)
INNOMCP จัดเตรียมชุดเครื่องมือ MCP ที่เชื่อมต่อ API ภาครัฐดังนี้:
1. **TMD/NWP Weather API** — พยากรณ์อากาศ จากกรมอุตุนิยมวิทยา  
   ใช้ดึงข้อมูลสภาพอากาศปัจจุบัน พยากรณ์ล่วงหน้า และคำเตือนภัย
2. **Thai Geo Tool** — ข้อมูลภูมิศาสตร์  
   ค้นหาพิกัด ขอบเขตการปกครอง และข้อมูลแผนที่ภาษาไทย
3. **Evidence Stats** — ข้อมูลสถิติ  
   เข้าถึงฐานข้อมูลสถิติทางการ เช่น ประชากร เศรษฐกิจ สาธารณสุข
4. **Thai Knowledge Base** — ฐานความรู้ภาษาไทย  
   รวมกฎหมาย ระเบียบ งานวิจัย และองค์ความรู้ของหน่วยราชการ
