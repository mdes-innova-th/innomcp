    ## 🔧 จงพัฒนา Tools สำหรับ MCP

    ### 1. **ArchiveTool**
    - **API:** Internet Archive Open Data  
    - **Function:** ค้นหา metadata ของหนังสือ, เพลง, วิดีโอ, dataset  
    - **Use case:** ผู้ใช้ถาม “หาหนังสือเกี่ยวกับ AI ที่อยู่ใน archive.org” → Tool ดึง JSON metadata มาแสดง  
    - **Implementation:**  
    - Endpoint: `https://archive.org/advancedsearch.php?q=...&output=json`  
    - Node tool: `archiveTool(query)` → return list of items  

    ---

    ### 2. **NasaTool**
    - **API:** NASA Open Data  
    - **Function:** ดึงข้อมูลภาพถ่ายดาวเทียม, ภารกิจ, Astronomy Picture of the Day (APOD)  
    - **Use case:** “ขอดูภาพ APOD วันนี้” → Tool เรียก NASA API แล้วส่งกลับ URL ภาพ  
    - **Implementation:**  
    - Endpoint: `https://api.nasa.gov/planetary/apod?api_key=DEMO_KEY`  
    - Node tool: `nasaTool(endpoint, params)`  

    ---

    ### 3. **WeatherTool**
    - **API:** OpenWeather  
    - **Function:** พยากรณ์อากาศ, historical weather, current conditions  
    - **Use case:** “พรุ่งนี้กรุงเทพฝนตกไหม” → Tool ดึง forecast 5 วันมาแสดง  
    - **Implementation:**  
    - Endpoint: `https://api.openweathermap.org/data/2.5/forecast?q=Bangkok&appid=API_KEY`  
    - Node tool: `weatherTool(city, type)`  

    ---

    ### 4. **WorldBankTool**
    - **API:** World Bank Open Data  
    - **Function:** ดึงข้อมูลเศรษฐกิจ เช่น GDP, inflation, population  
    - **Use case:** “GDP ไทย 2024 เท่าไหร่” → Tool ดึง indicator `NY.GDP.MKTP.CD`  
    - **Implementation:**  
    - Endpoint: `http://api.worldbank.org/v2/country/TH/indicator/NY.GDP.MKTP.CD?format=json`  
    - Node tool: `worldBankTool(country, indicator)`  

    ---

    ### 5. **GovDataTool**
    - **API:** US Gov 18F / Data.gov  
    - **Function:** ดึงข้อมูลภาครัฐ เช่น census, transport, health  
    - **Use case:** “จำนวนประชากรสหรัฐล่าสุด” → Tool ดึง dataset จาก Data.gov  
    - **Implementation:**  
    - Endpoint: `https://api.data.gov/...`  
    - Node tool: `govDataTool(datasetId, params)`  

    ---

    ## 🛠 การออกแบบ Tools ให้ “มีประโยชน์กว่า CalculatorTool”
    - **CalculatorTool เดิม:** ทำแค่ basic math → ไม่คุ้มค่า  
    - **แนวทางใหม่:**  
    - รวม **calculation 	แทน CalculatorTool เดิม:
        Hybrid Approach Design
        • 	NewtonTool → ใช้สำหรับ symbolic math เช่น derivative, integration, simplify algebra
        • 	MathTool → ใช้สำหรับ numeric/statistics เช่น mean, stddev, matrix calc, unit conversion
    Flow ตัวอย่าง
        1. ผู้ใช้ถาม: "หาอนุพันธ์ของ x^2 + 3x"
        MCP เลือก NewtonTool > ส่งไป https://newton.now.sh/api/v2/derive/x^2+3x
        คืนค่า "2x+3"

        2. ผู้ใช้ถาม: "ค่าเฉลี่ยของ [2,4,6,8]"
        MCP เลือก MathTool>evaluate('mean([2,4,6,8])') 
        คืนค่า 5



    - ทำให้ Tools ไม่ใช่แค่ “ดึงข้อมูล” แต่ **ประมวลผล + สรุปผล**
