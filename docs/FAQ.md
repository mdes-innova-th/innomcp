# คำถามที่พบบ่อย (FAQ)

## ทั่วไป
**Q: INNOMCP คืออะไร?**  
A: ระบบปัญญาประดิษฐ์สำหรับหน่วยงานภาครัฐ พัฒนาโดยกระทรวงดิจิทัลเพื่อเศรษฐกิจและสังคม (MDES) ทำงานในรูปแบบ multi-agent อัจฉริยะ ช่วยงานเอกสาร วิเคราะห์ข้อมูล และตอบคำถามด้วยโมเดลภาษาไทยประสิทธิภาพสูง

**Q: ใช้งานได้ฟรีหรือไม่?**  
A: ฟรีสำหรับบุคลากรภาครัฐทุกสังกัด ระบบใช้โครงสร้างพื้นฐาน MDES Ollama ซึ่งไม่มีค่าบริการเพิ่มเติม และสามารถใช้งานได้ 24 ชั่วโมง

**Q: ข้อมูลที่พิมพ์ลงในระบบปลอดภัยหรือไม่?**  
A: ปลอดภัยสูงสุด ข้อมูลทั้งหมดถูกประมวลผลภายในเซิร์ฟเวอร์ของรัฐภายใต้การกำกับของ MDES ไม่มีการส่งออกไปยังบุคคลที่สาม และปฏิบัติตามมาตรฐานคุ้มครองข้อมูลส่วนบุคคล (PDPA) อย่างเคร่งครัด

**Q: INNOMCP แตกต่างจาก ChatGPT อย่างไร?**  
A: ระบบถูกพัฒนาแบบ open-source ภายใต้การควบคุมของรัฐโดยตรง ข้อมูลผู้ใช้จะไม่ถูกนำไปใช้เทรนโมเดลภายนอก สามารถปรับแต่ง agent ได้ตามภารกิจของหน่วยงาน และเชื่อมต่อกับฐานข้อมูลภายในประเทศได้โดยไม่ต้องพึ่งพาบริการจากต่างประเทศ

## การใช้งาน
**Q: ทำไม AI ตอบช้าในบางครั้ง?**  
A: โมเดลขนาดใหญ่ต้องการเวลาในการประมวลผล หากต้องการความเร็วที่สูงขึ้น แนะนำให้เปลี่ยนโมเดลเป็นรุ่นเล็ก เช่น qwen2.5:7b ผ่านเมนู Settings (⚙️) โดยลดเวลาตอบลงได้มากกว่า 50%

**Q: รองรับการแนบไฟล์อะไรบ้าง?**  
A: รองรับไฟล์ PDF, Word, Excel, CSV, รูปภาพ และไฟล์เสียง ระบบจะใช้ Whisper ถอดความจากเสียงโดยอัตโนมัติ และประมวลผลเอกสารให้เนื้อหาพร้อมสำหรับการถาม-ตอบ

**Q: บันทึกประวัติการสนทนาได้หรือไม่?**  
A: ได้ กดปุ่ม Export ในส่วนหัวของบทสนทนา เลือกรูปแบบที่ต้องการ ได้แก่ PDF, Markdown หรือ JSON เพื่อเก็บเป็นหลักฐานหรือใช้รายงาน

**Q: สามารถใช้งานบนมือถือหรือแท็บเล็ตได้หรือไม่?**  
A: ใช่ อินเทอร์เฟซถูกออกแบบแบบ Responsive รองรับทุกขนาดหน้าจอ และสามารถติดตั้งเป็น Progressive Web App (PWA) เพื่อความสะดวกเสมือนแอปพลิเคชัน

## ด้านเทคนิค
**Q: รองรับภาษาไทยและฟอนต์หรือไม่?**  
A: รองรับภาษาไทยอย่างสมบูรณ์ ใช้ฟอนต์ Noto Sans Thai และ Sarabun ที่ให้การแสดงผลชัดเจน UI ทั้งหมดเป็นภาษาไทย

**Q: มี API สำหรับนักพัฒนาหรือไม่?**  
A: มี API สำหรับเชื่อมต่อระบบภายนอก อ่านรายละเอียดการเรียกใช้ได้ในเอกสาร docs/API_REFERENCE.md

**Q: สามารถเพิ่ม AI model อื่นได้หรือไม่?**  
A: ได้ เข้าไปที่ Settings → Providers → Add Provider เพื่อเชื่อมต่อกับบริการอื่น เช่น Ollama, OpenClaude หรือ Custom API

**Q: รองรับการปรับแต่งให้ AI เรียนรู้ข้อมูลภายในหน่วยงานหรือไม่?**  
A: รองรับผ่านฟีเจอร์ RAG (Retrieval-Augmented Generation) และ Knowledge Base อัปโหลดเอกสารภายในเพื่อให้ AI ตอบโดยอ้างอิงข้อมูลจริงของหน่วยงานได้ทันที
