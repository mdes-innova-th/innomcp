# INNOMCP Node.js Services Reference (Thai/English)

| Service | Purpose | Key Methods | When Initialized |
|---------|---------|-------------|-------------------|
| **analyticsService** | เก็บรวบรวมและวิเคราะห์ข้อมูลการใช้งานแพลตฟอร์ม (Collects and analyzes platform usage data) | `trackEvent(eventName, payload)`, `getReports(filter)`, `aggregateDaily()` | รันครั้งเดียวตอนเซิร์ฟเวอร์เริ่มทำงาน (Initialized once at server startup) |
| **auditLogger** | บันทึกประวัติการกระทำเพื่อการตรวจสอบความปลอดภัยและปฏิบัติตามกฎหมาย (Logs all actions for security auditing and legal compliance) | `logAction(userId, action, details)`, `queryLogs(filter, limit)`, `exportLogs(format)` | เริ่มทำงานตั้งแต่เซิร์ฟเวอร์เริ่มต้น (Starts when server starts) |
| **backpressureHandler** | จัดการควบคุมอัตราการรับ-ส่งข้อมูลในสตรีม ป้องกันระบบล้น (Handles backpressure in stream processing to prevent system overload) | `applyBackpressure(stream)`, `monitorBuffer()`, `release()` | เริ่มต้นเมื่อมีการเชื่อมต่อสตรีม (Initialized per stream connection) |
| **cacheManager** | จัดเก็บและจัดการข้อมูลในหน่วยความจำแคชเพื่อเพิ่มความเร็ว (Manages in-memory caching for faster data retrieval) | `get(key)`, `set(key, value, ttl)`, `invalidate(key)`, `clearAll()` | รันเมื่อเซิร์ฟเวอร์เริ่ม และใช้ร่วมกันทั้งแอป (Runs at startup, shared across the app) |
| **contextManager** | บริหารจัดการบริบทการสนทนาและสถานะของเซสชัน (Manages conversation context and session state) | `getContext(sessionId)`, `updateContext(sessionId, delta)`, `clearContext(sessionId)` | สร้างเมื่อมีเซสชันใหม่ (Created per session) |
| **eventBus** | ระบบพับลิช/ซับสไคร์สำหรับเหตุการณ์ภายในแอปพลิเคชัน (Publish/subscribe system for internal events) | `on(event, handler)`, `emit(event, data)`, `off(event, handler)` | เริ่มทำงานครั้งเดียวตอนเริ่มต้น (Initialized once at startup) |
| **healthAggregator** | รวบรวมสถานะความพร้อมของส่วนประกอบต่างๆ และส่งเป็นรายงาน (Aggregates health status of all components and provides reports) | `checkAll()`, `getHealthReport()`, `registerCheck(name, checkFn)` | เริ่มต้นพร้อมเซิร์ฟเวอร์ (Started at server startup) |
| **mdesModelCache** | แคชโมเดลเฉพาะของ MDES เพื่อลดเวลาโหลดและเพิ่มประสิทธิภาพ (Caches MDES-specific AI models to reduce load times and improve performance) | `cacheModel(modelId, model)`, `getModel(modelId)`, `evictModel(modelId)` | โหลดเมื่อเริ่มต้น หรือเมื่อมีการร้องขอครั้งแรก (Loaded at startup or on first request) |
| **metricsCollector** | เก็บรวบรวมเมตริกประสิทธิภาพการทำงาน (Collects performance metrics such as response times, throughput) | `recordMetric(name, value, tags)`, `getMetricsSnapshot()`, `startTimer()` | เริ่มต้นอัตโนมัติเมื่อเซิร์ฟเวอร์เริ่ม (Automatically starts at server startup) |
| **modelSuggestionService** | แนะนำโมเดล AI ที่เหมาะสมสำหรับแต่ละคำขอ (Recommends the most suitable AI model for each request) | `suggestModel(input, context)`, `learnFromFeedback(modelId, success)`, `getModelStats()` | เริ่มต้นตอนเซิร์ฟเวอร์เริ่ม; ทำงานเมื่อมีคำขอ (Starts at server start; used on each request) |
| **notificationService** | ส่งการแจ้งเตือนผ่านช่องทางต่างๆ เช่น อีเมล, ข้อความ (Sends notifications via email, push, etc.) | `send(to, message, channel)`, `registerDevice(userId, deviceToken)`, `getTemplates()` | เริ่มทำงานเมื่อเซิร์ฟเวอร์เริ่ม (Initialized at server startup) |
| **providerFailover** | จัดการการสลับไปยังผู้ให้บริการ AI สำรองเมื่อผู้ให้หลักมีปัญหา (Manages failover between primary and backup AI providers) | `registerProvider(name, config)`, `handleFailover(error)`, `getCurrentProvider()` | เริ่มต้นพร้อมเซิร์ฟเวอร์ (Initialized at startup) |
| **providerManager** | จัดการรายชื่อและสถานะของผู้ให้บริการ AI ทั้งหมด (Manages the list and status of all available AI providers) | `getProviders()`, `addProvider(config)`, `removeProvider(name)`, `checkAvailability(name)` | เริ่มต้นครั้งเดียวตอนเริ่มระบบ (Initialized once at startup) |
| **responseFormatter** | จัดรูปแบบคำตอบจาก AI ให้สวยงามและรองรับรูปแบบต่างๆ (Formats AI responses for output, e.g., markdown, plain text) | `formatResponse(raw, options)`, `applyTemplate(template, data)`, `toMarkdown(text)` | ทำงานตามคำขอ (Used per request) |
| **sessionService** | บริหารจัดการเซสชันผู้ใช้งาน (Manages user sessions including creation and termination) | `createSession(userId)`, `getSession(sessionId)`, `destroySession(sessionId)`, `extendSession(sessionId)` | เริ่มต้นครั้งเดียวตอนเริ่มเซิร์ฟเวอร์ (Initialized once at server startup) |
| **sessionStore** | จัดเก็บข้อมูลเซสชันในหน่วยความจำหรือฐานข้อมูลภายนอก (Stores session data in memory or external store like Redis) | `get(sid)`, `set(sid, data, expiry)`, `delete(sid)`, `all()` | เริ่มต้นครั้งเดียวตอนเริ่มเซิร์ฟเวอร์ (Initialized once at startup) |
| **streamManager** | จัดการสตรีมข้อมูลเรียลไทม์สำหรับการตอบกลับแบบ streaming (Manages real-time streaming of AI responses) | `startStream(sessionId, stream)`, `writeChunk(chunk)`, `endStream()`, `onError()` | สร้างขึ้นเมื่อมีการร้องขอ streaming (Created per streaming request) |
| **thaiGovtTools** | เชื่อมต่อและเรียกใช้เครื่องมือของหน่วยงานภาครัฐไทย (Integrates and invokes tools provided by Thai government agencies) | `registerTool(name, fn)`, `callTool(name, params)`, `listTools()` | เริ่มต้นตอนเซิร์ฟเวอร์เริ่ม (Initialized at server startup) |
| **thaiIntentRouter** | จำแนกและส่งต่อเจตนาของผู้ใช้ไปยังตัวจัดการที่เหมาะสม (Classifies and routes user intents to appropriate handlers) | `routeIntent(input, context)`, `registerHandler(intent, handler)`, `getIntents()` | เริ่มต้นตอนเซิร์ฟเวอร์เริ่ม (Initialized at startup) |
| **thaiNLPEnhancer** | ปรับปรุงประสิทธิภาพการประมวลผลภาษาไทย เช่น ตัดคำ, ทำความสะอาด (Enhances Thai NLP processing: tokenization, cleaning, normalization) | `enhance(text)`, `preprocessThai(input)`, `correctSpelling(text)` | เริ่มต้นครั้งเดียวตอนเริ่มเซิร์ฟเวอร์ (Initialized once at startup) |
| **thaiNLPService** | บริการหลักด้านการประมวลผลภาษาธรรมชาติภาษาไทย (Core Thai NLP service: tokenization, POS tagging, NER, sentiment) | `tokenize(text)`, `posTag(tokens)`, `ner(text)`, `sentiment(text)` | เริ่มต้นครั้งเดียวตอนเริ่มเซิร์ฟเวอร์ (Initialized once at startup) |
| **toolExecutor** | รับผิดชอบการทำงาน (Execute) ของเครื่องมือต่างๆ ตามคำขอ (Executes registered tools based on plan from AI) | `execute(toolName, args)`, `getAvailableTools()`, `validateArgs(schema, args)` | เริ่มต้นตอนเซิร์ฟเวอร์เริ่ม (Initialized at startup) |
| **workspaceService** | จัดการพื้นที่ทำงานของผู้ใช้ เช่น ไฟล์, เอกสาร และสถานะ (Manages user workspace: files, documents, and multi-step state) | `createWorkspace(userId, name)`, `getWorkspace(id)`, `updateWorkspace(id, delta)`, `deleteWorkspace(id)` | สร้างเมื่อผู้ใช้เริ่มใช้งานครั้งแรก หรือตามคำขอ (Created per user on first use or per request) |
| **wsEnhancer** | ปรับแต่งการเชื่อมต่อ WebSocket เช่น เพิ่ม compression, authentication (Enhances WebSocket connections with compression, authentication, logging) | `enhance(socket)`, `applyMiddleware(ws, middlewares)`, `authenticate(token)` | ทำงานทุกครั้งที่มีการอัปเกรดเป็น WebSocket (Applied on each WebSocket upgrade) |
