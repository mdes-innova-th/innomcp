# ประวัติการอัปเดต (Changelog)

บันทึกการเปลี่ยนแปลงสำคัญของโปรเจกต์ InnoMCP

## Version 2026.1.0 (Current)
**"Update"**
*   **[Documentation]**: ยกเครื่องเอกสารใหม่ทั้งหมด 10 หมวดหมู่ (Thai Language)
*   **[Core]**: อัปเกรด Node.js Backend ให้รองรับ Microservices
*   **[Tools]**: เพิ่ม MCP Tools ชุดใหม่ (OCR, PDF Reader)
*   **[UI]**: ปรับปรุงหน้า Chat UI ให้ตอบสนองไวขึ้น 200%

## Version 2025.12.0
*   เริ่มต้นพัฒนาโปรเจกต์
*   วางโครงสร้าง Next.js และ MariaDB
