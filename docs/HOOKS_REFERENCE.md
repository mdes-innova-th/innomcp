# INNOMCP React Hooks Reference
เอกสารอ้างอิง React Hooks สำหรับระบบ INNOMCP

| Hook                    | Returns<br>(EN / TH)                                                                                                              | When to use<br>(EN / TH)                                                                                                                         | Dependencies<br>(EN / TH)                                                                                                                                        |
|-------------------------|-----------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `useAgentCount`         | **EN:** Number of active AI agents currently registered in the MCP Hub.<br>**TH:** จำนวน AI agent ที่ลงทะเบียนและพร้อมใช้งานใน MCP Hub ในขณะนั้น           | **EN:** Display live agent statistics on dashboards, header badges, or status panels.<br>**TH:** แสดงสถิติ agent แบบเรียลไทม์บนแดชบอร์ด ป้ายสถานะ หรือแผงข้อมูล | **EN:** `useContext(AgentContext)`, `useState`, `useEffect`<br>**TH:** `useContext(AgentContext)`, `useState`, `useEffect`                                       |
| `useAgentStats`         | **EN:** Object with detailed agent metrics (uptime, request count, error rate, etc.).<br>**TH:** ออบเจ็กต์ที่บรรจุข้อมูลสถิติของ agent (เวลาทำงาน, จำนวนคำขอ, อัตราข้อผิดพลาด ฯลฯ) | **EN:** Admin monitoring, performance analysis, selecting the best provider.<br>**TH:** ใช้ในหน้าแอดมินสำหรับตรวจสอบประสิทธิภาพและเลือก provider ที่ดีที่สุด          | **EN:** `useContext(AgentStatsContext)`, `useState`, `useCallback`<br>**TH:** `useContext(AgentStatsContext)`, `useState`, `useCallback`                         |
| `useAutoResize`         | **EN:** Ref to attach to a `<textarea>`, automatically adjusts height based on content.<br>**TH:** Ref สำหรับผูกกับ `<textarea>` เพื่อปรับความสูงอัตโนมัติตามเนื้อหา | **EN:** Chat input, comment boxes, any expandable text field.<br>**TH:** ช่องป้อนข้อความแชท กล่องคอมเมนต์ หรือฟิลด์ข้อความที่ขยายได้                     | **EN:** `useRef`, `useEffect`, `useCallback`<br>**TH:** `useRef`, `useEffect`, `useCallback`                                                                     |
| `useChatErrorHandler`   | **EN:** Object `{ error, setError, clearError, showErrorToast }`.<br>**TH:** ออบเจ็กต์ `{ error, setError, clearError, showErrorToast }`                | **EN:** Centralised chat error management, display user‑friendly error messages (Thai language).<br>**TH:** จัดการข้อผิดพลาดในแชทแบบรวมศูนย์ แสดงข้อความเป็นภาษาไทย | **EN:** `useState`, `useCallback`, `useNotifications` (for toasts)<br>**TH:** `useState`, `useCallback`, `useNotifications` (สำหรับ toast)                        |
| `useChatMessages`       | **EN:** `{ messages, addMessage, updateMessage, deleteMessage, isLoading }`.<br>**TH:** `{ messages, addMessage, updateMessage, deleteMessage, isLoading }` | **EN:** Manage chat history in a React component; supports optimistic updates and streaming.<br>**TH:** จัดการประวัติแชทในคอมโพเนนต์ รองรับการอัปเดตแบบ optimistic และสตรีมมิ่ง | **EN:** `useState`, `useCallback`, `useMDESOllama` (for AI responses)<br>**TH:** `useState`, `useCallback`, `useMDESOllama` (สำหรับตอบกลับจาก AI)                  |
| `useChatState`          | **EN:** Unified chat state `{ messages, input, isStreaming, error, connectionStatus }`.<br>**TH:** สถานะแชทที่รวมไว้ `{ messages, input, isStreaming, error, connectionStatus }` | **EN:** Top‑level chat container; passes state down to child components.<br>**TH:** คอนเทนเนอร์ระดับบนสุดของแชท ส่งต่อสถานะไปยังคอมโพเนนต์ลูก           | **EN:** `useChatMessages`, `useChatErrorHandler`, `useMDESOllama`<br>**TH:** `useChatMessages`, `useChatErrorHandler`, `useMDESOllama`                           |
| `useCopyToClipboard`    | **EN:** `{ copiedText, copyToClipboard, isCopied }`.<br>**TH:** `{ copiedText, copyToClipboard, isCopied }`                                          | **EN:** Copy message content or code blocks to clipboard with feedback.<br>**TH:** คัดลอกข้อความหรือโค้ดไปยังคลิปบอร์ด พร้อมแสดงสถานะ                 | **EN:** `useState`, `useCallback`, `navigator.clipboard`<br>**TH:** `useState`, `useCallback`, `navigator.clipboard`                                             |
| `useDebounce`           | **EN:** Debounced value that updates after a specified delay.<br>**TH:** ค่าแบบดีบาวนซ์ที่อัปเดตเมื่อพ้นระยะเวลาที่กำหนด                          | **EN:** Search inputs, filtering, any action that should wait for user to stop typing.<br>**TH:** ช่องค้นหา การกรอง หรือการทำงานที่ต้องรอให้ผู้ใช้หยุดพิมพ์ | **EN:** `useState`, `useEffect`<br>**TH:** `useState`, `useEffect`                                                                                             |
| `useKeyboardShortcuts`  | **EN:** `void` – registers global keyboard shortcut listeners.<br>**TH:** `void` – ลงทะเบียนตัวรับฟังคีย์ลัดระดับโกลบอล                     | **EN:** Implement app‑wide shortcuts (e.g., Ctrl+Enter to send, Escape to clear).<br>**TH:** ใช้สร้างทางลัดทั่วทั้งแอป เช่น Ctrl+Enter ส่งข้อความ Escape ล้างข้อความ | **EN:** `useEffect`, `useCallback`<br>**TH:** `useEffect`, `useCallback`                                                                                       |
| `useLocalStorageState`  | **EN:** `[storedValue, setValue]` – behaves like `useState` but persists to `localStorage`.<br>**TH:** `[storedValue, setValue]` คล้าย `useState` แต่เก็บใน `localStorage` | **EN:** Persist user preferences (theme, language) across sessions; offline‑first data.<br>**TH:** เก็บค่ากำหนดผู้ใช้ (ธีม ภาษา) ให้คงอยู่ข้ามเซสชัน หรือข้อมูลแบบออฟไลน์ก่อน | **EN:** `useState`, `useEffect`<br>**TH:** `useState`, `useEffect`                                                                                             |
| `useMDESChatShortcuts`  | **EN:** Custom shortcut map specific to MDES chat interface (e.g., Thai‑specific key combinations).<br>**TH:** แผนที่ทางลัดที่ปรับแต่งสำหรับอินเทอร์เฟซแชทของ MDES (เช่น การผสมแป้นภาษาไทย) | **EN:** MDES chat UI when the default shortcuts need to be overridden or extended.<br>**TH:** ใช้ใน UI แชท MDES เมื่อต้องการแทนที่หรือขยายทางลัดมาตรฐาน    | **EN:** `useKeyboardShortcuts`, `useCallback`<br>**TH:** `useKeyboardShortcuts`, `useCallback`                                                                 |
| `useMDESOllama`         | **EN:** `{ sendMessage, streamResponse, abort, isConnected, modelList }` – client for self‑hosted Ollama.<br>**TH:** `{ sendMessage, streamResponse, abort, isConnected, modelList }` – ไคลเอนต์สำหรับ Ollama ที่โฮสต์เอง | **EN:** Any component that needs to call the MDES Ollama API for chat completions.<br>**TH:** คอมโพเนนต์ที่ต้องการเรียก API ของ MDES Ollama เพื่อสร้างข้อความตอบกลับ | **EN:** `useState`, `useCallback`, `useMDESProviderHealth`, fetch/streaming<br>**TH:** `useState`, `useCallback`, `useMDESProviderHealth`, fetch/streaming       |
| `useMDESProviderHealth` | **EN:** `{ isHealthy, lastChecked, checkHealth, healthHistory }`.<br>**TH:** `{ isHealthy, lastChecked, checkHealth, healthHistory }`                  | **EN:** Health status badge, automatic failover, background health polling.<br>**TH:** ป้ายสถานะความพร้อม ระบบสำรองอัตโนมัติ การตรวจสุขภาพเบื้องหลัง    | **EN:** `useState`, `useEffect`, `useCallback`<br>**TH:** `useState`, `useEffect`, `useCallback`                                                               |
| `useMDESTheme`          | **EN:** `{ theme, setTheme, toggleTheme }` – MDES corporate branding (colors, fonts, dark/light).<br>**TH:** `{ theme, setTheme, toggleTheme }` – ธีมองค์กร MDES (สี ฟอนต์ โหมดมืด/สว่าง) | **EN:** Apply MDES visual identity across the portal; respect user preference or system theme.<br>**TH:** ใช้ปรับเอกลักษณ์ภาพของ MDES บนพอร์ทัล และเคารพการตั้งค่าธีมของผู้ใช้ | **EN:** `useLocalStorageState`, `useEffect`, MDES design tokens<br>**TH:** `useLocalStorageState`, `useEffect`, MDES design tokens                              |
| `useMessageGrouping`    | **EN:** `{ groupedMessages: Map<groupKey, Message[]> }` – groups messages by date, sender, or conversation.<br>**TH:** `{ groupedMessages: Map<groupKey, Message[]> }` – จัดกลุ่มข้อความตามวันที่ ผู้ส่ง หรือการสนทนา | **EN:** Display chat history with date separators, thread view, or agent‑specific lanes.<br>**TH:** แสดงประวัติแชทที่มีตัวแบ่งวันที่ มุมมองเธรด หรือช่องแยกตาม agent | **EN:** `useMemo`<br>**TH:** `useMemo`                                                                                                                         |
| `useNotifications`      | **EN:** `{ notifications, addNotification, removeNotification, clearAll }`.<br>**TH:** `{ notifications, addNotification, removeNotification, clearAll }` | **EN:** In‑app toast/notification center for errors, warnings, or success messages.<br>**TH:** ศูนย์การแจ้งเตือนในแอป สำหรับข้อผิดพลาด คำเตือน หรือข้อความสำเร็จ | **EN:** `useState`, `useCallback`<br>**TH:** `useState`, `useCallback`                                                                                         |
| `useOfflineSync`        | **EN:** `{ isOnline, pendingQueue, syncNow, lastSyncTime }`.<br>**TH:** `{ isOnline, pendingQueue, syncNow, lastSyncTime }`                          | **EN:** Handle offline chat input queue and automatic sync when connectivity resumes.<br>**TH:** จัดการคิวข้อความตอนออฟไลน์และซิงค์อัตโนมัติเมื่อเชื่อมต่ออีกครั้ง | **EN:** `useState`, `useEffect`, `navigator.onLine`, IndexedDB (optional)<br>**TH:** `useState`, `useEffect`, `navigator.onLine`, IndexedDB (ถ้ามี)               |
| `usePrevious`           | **EN:** The previous value of a given state or prop.<br>**TH:** ค่าก่อนหน้าของ state หรือ prop ที่ระบุ                                    | **EN:** Compare changes (e.g., detect new messages, scroll only when message list grows).<br>**TH:** เปรียบเทียบการเปลี่ยนแปลง เช่น ตรวจจับข้อความใหม่ เลื่อนหน้าจอเมื่อรายการข้อความเพิ่มขึ้น | **EN:** `useRef`, `useEffect`<br>**TH:** `useRef`, `useEffect`                                                                                                 |
| `useProviderHealth`     | **EN:** `{ providers, checkProvider, healthStatus }` – generic version for multiple LLM providers.<br>**TH:** `{ providers, checkProvider, healthStatus }` – เวอร์ชันทั่วไปสำหรับผู้ให้บริการ LLM หลายราย | **EN:** Multi‑provider dashboards, automatic routing based on health.<br>**TH:** แดชบอร์ดผู้ให้บริการหลายราย ระบบเลือกเส้นทางอัตโนมัติตามสถานะความพร้อม | **EN:** `useState`, `useCallback`, `useEffect`<br>**TH:** `useState`, `useCallback`, `useEffect`                                                               |
| `useScrollBehavior`     | **EN:** `{ containerRef, scrollToBottom, isAtBottom, scrollToMessage }`.<br>**TH:** `{ containerRef, scrollToBottom, isAtBottom, scrollToMessage }`  | **EN:** Advanced scroll control: sticky bottom, smooth scroll, jump to a specific message.<br>**TH:** ควบคุมการเลื่อนขั้นสูง: ติดด้านล่าง เลื่อนนุ่มนวล กระโดดไปข้อความที่ระบุ | **EN:** `useRef`, `useCallback`, `useEffect`<br>**TH:** `useRef`, `useCallback`, `useEffect`                                                                   |
| `useScrollToBottom`     | **EN:** `scrollToBottomRef` – a ref to assign to the scroll container; automatically scrolls when children change.<br>**TH:** `scrollToBottomRef` – ref สำหรับผูกกับคอนเทนเนอร์เลื่อน จะเลื่อนลงอัตโนมัติเมื่อเนื้อหาเปลี่ยน | **EN:** Chat message list; keep newest messages visible.<br>**TH:** รายการข้อความแชท ให้เห็นข้อความล่าสุดเสมอ                              | **EN:** `useRef`, `useEffect`<br>**TH:** `useRef`, `useEffect`                                                                                                 |
| `useWindowSize`         | **EN:** `{ width, height }` – current browser window dimensions.<br>**TH:** `{ width, height }` – ขนาดหน้าต่างเบราว์เซอร์ในปัจจุบัน        | **EN:** Responsive layouts, conditionally render components based on screen size.<br>**TH:** เลย์เอาต์ที่ตอบสนองตามขนาดหน้าจอ เรนเดอร์คอมโพเนนต์ตามเงื่อนไขขนาด | **EN:** `useState`, `useEffect`, `resize` event<br>**TH:** `useState`, `useEffect`, อีเวนต์ `resize`                                                             |
