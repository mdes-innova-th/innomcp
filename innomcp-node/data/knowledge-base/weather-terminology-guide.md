# คู่มืออากาศ — อธิบายศัพท์พยากรณ์อากาศ

## โอกาสฝน (Rain Probability)

โอกาสฝน 40% หมายถึง มีความน่าจะเป็น 40% ที่จะมีฝนตกในพื้นที่และช่วงเวลานั้น
ไม่ได้หมายความว่า 40% ของพื้นที่จะมีฝน แต่หมายถึงความน่าจะเป็นของการเกิดฝน

### ระดับโอกาสฝน

| ช่วง | ความหมาย |
|------|----------|
| 0-20% | ฝนตกน้อยมาก หรือไม่มีฝน |
| 20-40% | มีโอกาสฝนเล็กน้อย |
| 40-60% | มีโอกาสฝนปานกลาง |
| 60-80% | มีโอกาสฝนค่อนข้างมาก |
| 80-100% | ฝนตกแน่นอน หรือเกือบแน่นอน |

## อุณหภูมิ (Temperature)

อุณหภูมิที่รายงานในพยากรณ์อากาศ วัดจากสถานีตรวจอากาศของกรมอุตุนิยมวิทยา (TMD)
เป็นอุณหภูมิอากาศในร่ม วัดที่ความสูง 1.5 เมตรจากพื้นดิน

### ค่าเฉลี่ยอุณหภูมิประเทศไทย

- ฤดูร้อน (มี.ค.-พ.ค.): 33-40°C
- ฤดูฝน (มิ.ย.-ต.ค.): 28-35°C
- ฤดูหนาว (พ.ย.-ก.พ.): 18-30°C

## แหล่งข้อมูลอากาศ

- **TMD** (กรมอุตุนิยมวิทยา) — แหล่งข้อมูลหลักสำหรับพยากรณ์อากาศประเทศไทย
- **NWP** (Numerical Weather Prediction) — แบบจำลองเชิงตัวเลขสำหรับพยากรณ์ล่วงหน้า
- **Station Data** — ข้อมูลสถานีจริง ณ เวลาปัจจุบัน

## การอ่านพยากรณ์อากาศ

พยากรณ์อากาศจะแสดง:
1. **อุณหภูมิสูงสุด/ต่ำสุด** — ช่วงอุณหภูมิในวันนั้น
2. **โอกาสฝน** — ความน่าจะเป็นของฝน (%)
3. **ปริมาณฝน** — ปริมาณฝนที่คาดว่าจะตก (มม.)
4. **ความเร็วลม** — ลม (กม./ชม.)
5. **ความชื้นสัมพัทธ์** — ความชื้นในอากาศ (%)
